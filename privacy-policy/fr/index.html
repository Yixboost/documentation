    Politique de Confidentialité (Français)

    Chez Yixboost NL, nous prenons votre vie privée au sérieux. Cette politique de confidentialité décrit les données que nous collectons, comment nous les utilisons et les choix que vous avez concernant vos données.

    1. Collecte des Données

    Google Analytics

    Nous utilisons Google Analytics pour analyser l'utilisation de notre site Web. Google Analytics collecte des données telles que votre adresse IP, le type de votre navigateur, les pages que vous visitez, le temps que vous passez sur le site et d'autres données associées. Ces informations nous aident à comprendre comment les visiteurs interagissent avec le site afin de l'améliorer.

    Google Analytics peut utiliser des cookies pour collecter des données. Pour plus d'informations sur la façon dont Google traite vos données, vous pouvez consulter la politique de confidentialité de Google.

    Services Github

    Nous utilisons les services de Github pour héberger certaines parties de notre site Web et d'autres services. Github peut collecter des données telles que vos interactions avec le site, et nous suivons leur politique de confidentialité lors de l'utilisation de leur plateforme. Pour plus d'informations, vous pouvez consulter la politique de confidentialité de Github.

    2. Utilisation des Données

    Les données que nous collectons via Google Analytics sont utilisées uniquement à des fins analytiques, pour améliorer les performances de notre site Web et comprendre comment il est utilisé.

    Les données collectées par Github sont utilisées pour fournir les services offerts par leur plateforme.

    3. Vos Choix

    Vous pouvez choisir de bloquer les cookies de Google Analytics en ajustant les paramètres de votre navigateur. Cela peut toutefois affecter la fonctionnalité du site. Pour plus d'informations sur la désactivation des cookies, vous pouvez consulter les paramètres de votre navigateur.

    Vous pouvez également gérer vos préférences en matière de cookies via les paramètres de votre navigateur.

    4. Modifications de cette Politique de Confidentialité

    Nous nous réservons le droit de mettre à jour cette politique de confidentialité de temps en temps. Toute modification sera publiée sur cette page, et la date de la dernière mise à jour sera indiquée en bas de la politique.

    5. Contact

    Si vous avez des questions concernant cette politique de confidentialité ou la manière dont nous traitons vos données, vous pouvez nous contacter via la page de contact sur notre site Web.
